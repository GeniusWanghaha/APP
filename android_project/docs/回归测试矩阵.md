# 回归测试矩阵（软件侧）

## 1. 协议与链路
- 分包重组乱序恢复（`FrameSegmentAssembler`）
- CRC 校验失败丢弃
- FrameID 跳变检测与短缺口补点
- 长缺口下高阶指标门控降级

## 2. 时间轴重建
- ECG 固定周期时间轴
- PPG 相位/延时补偿
- BLE 抖动时“到包时间 != 采样时间”保护
- 会话跨窗口拼接连续性

## 3. 指标与门控
- HR / SpO2 / PTT 基础指标
- HRV（SDNN / RMSSD / pNN50）在 SQI 达标时输出
- PAT/PTT/PWTT 仅在跨模态一致时输出
- 异常提示：心动过速/过缓、节律不齐、房颤风险、疑似早搏

## 4. 错误与边界
- BLE 权限拒绝
- 设备断连自动收敛为安全状态
- 低质量信号下阻止高阶结论
- AI 文案边界（提示/复测建议，不确诊）

## 5. 数据与导出
- Session 元数据写入与结束收口
- Session 事件日志写入
- CSV/JSON 导出可用性
- 追溯字段（sessionId/algorithmVersion/modelVersion）完整性
