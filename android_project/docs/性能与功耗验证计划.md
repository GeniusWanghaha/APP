# 性能与功耗验证计划

## 软件性能目标
- 端到端显示延迟 < 300ms
- 30~60 秒演示会话无卡顿
- 长会话（10 分钟）内存稳定，不持续增长

## 软件验证项（可在无硬件时先完成）
- 单元测试：时间轴重建、门控逻辑、风险提示
- 构建验证：`assembleDemoDebug` / `assembleContestDebug`
- Lint 验证：Compose 与 Kotlin 规范

## 需硬件实测项（待后续连机）
- 不同 LED 电流下功耗曲线
- 不同 BLE 参数下发热与丢包率
- 手机机型差异下 OTG 供电稳定性

## 建议记录格式
- 机型 / 固件版本 / APP 版本
- 采样参数（ECG/PPG 采样率、LED 电流、MTU）
- 平均电流、峰值温升、丢包率、SQI 均值
